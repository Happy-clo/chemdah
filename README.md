# Chemdah
Chemdah 是免费的，但我们不提供 jar 文件，你可以通过以下步骤自行构建插件。

**Windows 平台**
```shell
gradlew.bat build
```

**macOS/Linux 平台**
```shell
./gradlew clean build
```

**JavaDoc**：https://jd.ptms.ink/chemdah
